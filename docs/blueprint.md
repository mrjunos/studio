# **App Name**: BrewBooks MVP

## Core Features:

- Dashboard Metrics: Display key metrics such as Total Sales, Other Income, and Top Selling Products.
- Product Management: Allow adding, editing, and deleting products, and manage categories, price and current stock.
- Sales Registration: Allow quick sales registration and automatic reduction of inventory of sold products
- Inventory Adjustment: List inventory adjustments with product selection and quantity modification using positive or negative values.
- Other Income Registration: Allow other income tracking.
- Category Suggestion: When creating a product, provide AI-driven category suggestions based on product name using a tool. Suggest from the given valid options: Drinks, Food, Merchandise.

## Style Guidelines:

- Primary color: Medium brown (#A0522D) to reflect the coffee theme.
- Background color: Very light brown (#F5F5DC), almost white.
- Accent color: Muted orange (#DEB887) to create highlights and CTAs.
- Secondary accent color: Soft green (#8FBC8F) for subtle UI elements.
- Highly legible sans-serif font.
- Thematic icons (e.g., coffee bean for products, till for sales).
- Use a clear, fixed sidebar for main navigation.
- Provide clear visual feedback using subtle animations.
